# narikiri · なりきり

一个把你的内容"换衣服"穿的 Claude Code / Claude Agent Skill。

输入一段话 → 选一个风格（或描述自定义风格） → 输出该风格下的改写版本。

保留**意思不变**，只换**表达的声音**。

---

## 它能做什么

把同一个意思，用不同的"声音"说出来。预置 7 大风格家族，涵盖常用场景；也支持用户用自然语言描述的自定义风格。

### 支持的风格

| 家族 | 具体风格 |
|---|---|
| **汉语官方体** | 体制内汇报风、新闻联播风、领导讲话风、八股文 |
| **学术体** | 毕业答辩风、学术论文风、文献综述风 |
| **阴阳怪气 / 毒舌** | 阴阳怪气风、毒舌评论风、反讽风 |
| **互联网风** | 小红书风、微商风、互联网黑话、推特风、Reddit 风、LinkedIn humblebrag |
| **文学体** | 鲁迅风、文言文风、朋友圈文艺风、煽情鸡汤风 |
| **商务体** | 商务邮件（中/英）、律师函、客服道歉、LinkedIn |
| **日语体** | 敬語 / 商務日語 / ビジネスメール / 論文 |
| **自定义** | "像我妈发家庭群"、"像 GPT-3 刚出来时那种"、任意描述 |

### 多语言

- **输入**：任意语言
- **输出**：默认与输入相同；可用户指定（"改成英文"、"日本語で"）
- **跨语言 + 跨风格**：如"把这句中文改成 LinkedIn 英文风格"，原生支持

### 典型触发

- "把这段话改成体制内汇报风"
- "帮我写得更像毕业答辩"
- "用阴阳怪气的语气回一下这句"
- "make this sound like a LinkedIn post"
- "把我的中文想法改成日语敬语版本"
- "同一句话，给我写 3 种风格版本"

## 安装

Claude Code 和 Codex CLI **使用同一份 SKILL.md 格式**（YAML frontmatter + `references/`），一次安装两端可用：

```bash
git clone <this-repo> ~/code/narikiri
cd ~/code/narikiri
./install.sh
```

`install.sh` 把 repo 软链到两端的 skills 目录：

- `~/.claude/skills/narikiri` → `<repo>`
- `~/.codex/skills/narikiri` → `<repo>`

之后改 SKILL.md 或 references，两端立即生效，无需重装。

### 手动安装

```bash
ln -s ~/code/narikiri ~/.claude/skills/narikiri
ln -s ~/code/narikiri ~/.codex/skills/narikiri
```

### 验证

新开 Claude Code 或 Codex 会话，自然语言触发即可（两端都靠 SKILL.md 的 `description` 字段自动发现）：

```

```

应输出：


## 输出格式

## 示例

**输入**：

> 这个项目做完了，效果不错，但有几个小问题。

**→ 体制内汇报风**：

> 一是项目整体推进平稳有序，各项任务基本落实到位，预期目标初步达成。
> 二是在项目推进过程中仍存在部分薄弱环节，需要在下一阶段持续研究完善。
> 下步将统筹谋划、精准发力，推动相关工作再上新台阶。

**→ 毕业答辩风**：

> 本项目已完成既定的核心目标，整体实施效果较好。需要指出的是，本研究在执行过程中仍存在若干局限：[具体问题]。这些问题将在后续工作中予以进一步研究和改进。恳请各位老师批评指正。

### 为什么按家族拆 reference

每个风格家族的典型词汇 / 句式差异很大（体制内 vs. 小红书几乎没有重合词汇），按需加载避免无关 context 进入 prompt。

---

## 核心原则

- **保留事实不篡改**。风格可以修饰语气，但不能把"没做完"改成"已完成"。
- **承诺完全的风格**。半官半俗读起来最假；要做就做全套。
- **尊重风格原生长度**。体制内汇报自然较长；推特必须压短。不机械保持原字数。
- **不说教**。用户要阴阳怪气，就认真给 ta 写好阴阳怪气版，不劝"这样不好"。
- **不伪造事实**。原文没数字 / 引用就不要编造，用"相关数据"等模糊占位。
- **多语言原生**。英文阴阳用"as per my last email"这种英语本土话术，不直译中文梗。

## 名字来源

**narikiri** = 日语「なりきり」，意为"完全变身成（某角色/某风格）"。日常用法常见于 cosplay、角色扮演、模仿秀场景 —— 精准对应本 skill 的"把文字变装成另一种风格"。
