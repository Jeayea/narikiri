# 阴阳怪气 / 毒舌 / 反讽体

核心特征：**表面合规 + 实质攻击**。字面单看没问题，放上下文和语气就带刺。

## 阴阳怪气的 6 个核心手法

1. **过度礼貌** — 超出场景所需的客气。"**麻烦您**帮我看一下" 在熟人语境里 = 刺。
2. **反向夸奖** — 夸奖的点和场景不匹配。出事之后夸"真是太努力了"。
3. **"开玩笑"挡箭牌** — 先攻击，再"哈哈我开玩笑的啦"给自己留台阶。
4. **反问** — "我有说错吗？" / "不然呢？" / "这不是很明显吗？"
5. **引用对方的话打对方** — "你不是说没问题的吗？" / "按你说的做不就好了？"
6. **夸张附和** — "好的好的好的好的" / "对对对对对" / "行行行"，字数越多越假。

## 子风格

### 1. 阴阳怪气风（隐刃流）

**特征**：刺藏在糖里。单句无攻击，合起来全是针。

**高频元素**：
- 引号把对方的词括起来："哦，原来你'很忙'啊"
- emoji：🙂 🤡 😅 🙃（与严肃事配对时几乎 100% 反讽）
- "辛苦了" / "真有你的" / "不愧是你"（配合失败语境）
- "牛 👍"（单字 + 敷衍 emoji）
- 单字回复："哦" / "嗯" / "。"

**例**：
- 原文："他又搞砸了，真服了。"
- 改写："哦，这次又'挑战自我'了是吧🙂 不愧是你，从不让人失望。真的，真·牛。👍"

### 2. 毒舌评论风（明刃流）

**特征**：直接开刺，不遮遮掩掩。像脱口秀吐槽、豆瓣短评、早期微博段子手。

**句式**：
- "……这算个什么……"
- "我建议你重新学一下 X。"
- "上一次看到这种操作还是在动物世界。"
- "这个水平，建议直接改行。"
- "标准就是这么被拉低的。"
- "就这？"

**例**：
- 原文："这个 PPT 做得不行。"
- 改写："这 PPT 什么水平啊，配色像 00 年代网吧开机动画，排版像我二姨的朋友圈。说实话，就这美学，建议直接打印出来挂冰箱上辟邪。"

### 3. 反讽 / sarcasm 风

**特征**：完全相反意思，靠上下文和语气传递。字面上看甚至是"真心夸"。

**模板**：
- "好极了。"（明显糟糕的事之后）
- "真是个好主意。"（糟糕主意之后）
- "谢谢啊，真的太感谢了。"（被坑之后）

**关键**：反讽必须依赖语境。没有前置情境时读者会当真。

## 英语 sarcastic 的对应手法

用户要英文阴阳怪气时，用英语本土话术，不要直译中文梗：

- "Per my last email ..." (= 我说过了，你没看)
- "As discussed ..." (= 已经定了，别重开)
- "Kindly see attached." (= 客气外壳，内心不耐烦)
- "Interesting." (单独一词 = 不认同)
- "Happy to jump on a call to clarify." (= 你自己再读一遍)
- "Circling back on this!" (= 第 N 次催你了)
- "Just a friendly reminder." (= 已经不 friendly 了)
- "Bless your heart." (美国南方 — 通常 = 你真蠢但我不说)

## 改写注意

- **不要写出人身攻击或歧视**。毒舌针对"行为 / 作品 / 选择"，不针对对方身份（性别、外貌、家庭背景）。
- **阴阳怪气有"智商下限"**。太明显的阴阳读起来很蠢。好的阴阳是"对方看了之后要想 3 秒才能确定你在阴他"。
- **反讽要有语境锚点**。孤立的"真厉害"不成反讽；必须配合"在 X 背景下"才能读出反讽。
- **用户指定程度**：如果用户说"阴阳轻一点"，用 1 个手法；"阴阳狠一点"，叠 3–4 个。
