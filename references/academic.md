# 学术体

核心特征：**明确限定** + **被动语态** + **前人研究锚定** + **保留不确定性** + **术语规范化**。追求"严谨到可以被引用"的质感。

## 共性特征

- **限定词密集**：在……条件下、就……而言、从……角度
- **被动语态**：被认为 / 被视为 / 可被理解为 / 常被引用为
- **动词学术化**：看 → 考察；想 → 论证；发现 → 揭示 / 指出；很多 → 为数不少 / 相当数量
- **条件化表达**：若……则……；在某种意义上；可以说
- **前人锚定**：诚如 X (年份) 所指出、参照 Y 的研究
- **避免绝对词**：从不说"绝对"、"肯定"、"一定"，改用"倾向于"、"在多数情况下"、"在一定程度上"
- **明示不确定性**：尚待进一步研究、仍有争议、有待验证

## 子风格

### 1. 毕业答辩风

**语境**：学生面对评委，自我谦抑 + 展示严谨 + 主动认错。

**开场模板**：
```
各位老师好，我是 XX 专业的 XX，我的论文题目是《XXX》。下面我从四个方面汇报：
一、研究背景与意义
二、研究方法
三、主要发现
四、不足与展望
```

**签名句式**：
- "本研究的主要贡献在于……"
- "需要指出的是，本研究存在以下局限：一是……二是……"
- "受限于样本量 / 时间 / 数据可得性，……"
- "下一步研究将……"
- "诚恳欢迎各位老师批评指正。"

**情绪 tenor**：克制、谦虚、但准备好为每个结论辩护。

**例**：
- 原文："我这个研究挺有意思的，结论就是 A 导致 B。"
- 改写：
  > 本研究在特定样本条件下，考察了 A 与 B 之间的关系。研究结果显示，A 对 B 具有显著的正向影响 (p < 0.05)。需要指出的是，受限于样本范围和变量控制，本结论的外部有效性仍有待进一步验证。

### 2. 学术论文风

**语境**：正式发表文体，无口语残留。

**签名句式**：
- "已有研究表明……然而，关于 X 的讨论仍然不足。"
- "本文在 Y 的基础上，进一步探讨了……"
- "结果表明，在控制了 Z 之后，X 对 Y 的影响仍然显著。"
- "这一发现与 A (year) 的观点一致 / 相左。"
- "综上所述，本文贡献体现在三个方面……"

**禁忌**：
- 不用感叹号
- 不用"我认为"（改用"本研究认为 / 本文指出"）
- 不用"非常"、"特别"、"超级"

### 3. 文献综述风

**特征**：以"已有研究"为主语，铺陈多方观点，最后点出 gap。

**模板**：
```
关于 X，已有研究大致形成三种观点。
一是 ……（作者 A, 年份；作者 B, 年份）。
二是 ……（作者 C, 年份）。
三是 ……
然而，上述研究尚未充分讨论 ……。本研究试图填补这一空白。
```

## 改写注意

- **如果原文没有引用 / 数据**，不要虚构。用"相关研究"、"已有文献"做占位，标注为待补引用。
- **保留不确定性**。学术文体的核心不是"显得很确定"，而是"清晰标注确定性的边界"。
- **术语不要乱用**。如果原文没说"因果"，不要改成"因果关系"（可能只是相关）。
- **毕业答辩风要有谦语**。"恳请"、"不足"、"欢迎指正"是必须的。
