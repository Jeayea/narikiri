# 商务体

## 1. 商务邮件风（中文）

**核心特征**：开头问候 + 说明来意 + 条理清晰 + 结尾礼貌 + 签名规范。

**模板**：
```
X 总 / X 老师 您好：

[开头] 感谢您……/ 打扰您……/ 承蒙……

[正文]
1. ……
2. ……
3. ……

如有任何问题，欢迎随时沟通。期待您的回复。

顺祝商祺，
XXX
[日期]
```

**高频句式**：
- "兹就 XX 事宜，特此沟通如下"
- "现将有关情况汇报如下"
- "烦请您……"
- "敬请核示"
- "请见附件"
- "如蒙回复，不胜感激"

## 2. 商务邮件风（英文）

**结构**：Greeting → Purpose → Body → Call to action → Sign-off

**Formal 程度分级**：

| Level | 开头 | 结尾 |
|---|---|---|
| Very formal | Dear Dr. Smith, | Yours sincerely, |
| Formal | Dear John, | Best regards, |
| Neutral | Hi John, | Best, |
| Casual | Hey John, | Cheers, |

**高频句式**：
- "I hope this email finds you well."
- "I'm writing to follow up on ..."
- "Just circling back on ..."
- "Please find attached ..."
- "Let me know if you have any questions."
- "Looking forward to your reply."

**Sign-offs 解码**（见 kukioyomu 的 cross-cultural ref）：Best regards 默认中性；Regards 偏冷；Cheers 偏友好。

## 3. 律师函风

**核心特征**：严格法言 + 事实分段 + 明确诉求 + 时限 + 法律后果警告。

**结构模板**：
```
XX 先生 / 女士：

本律师事务所接受 XX 的委托，就 贵方 YY 事宜，特此函告如下：

一、事实经过
[时间、地点、事件陈述]

二、贵方行为构成
[法律定性]，违反了《XX 法》第 X 条。

三、本律师代表委托人正式要求：
1. 贵方应于本函送达后 X 日内……
2. ……
3. ……

逾期未履行上述义务的，本律师将代表委托人依法采取包括但不限于提起诉讼、申请财产保全等一切法律措施，由此产生的一切法律后果由贵方自行承担。

特此函告。

XX 律师事务所
XX 律师（签字）
XXXX 年 X 月 X 日
```

**签名词汇**：
特此函告、依法、不再另行通知、一切法律后果、责令、书面答复、正式主张、保留追诉权利

## 4. 客服道歉风

**核心特征**：共情 + 担责 + 具体补偿 + 预防承诺 + 真诚落款。

**模板**：
```
尊敬的 XX 用户：

您好。非常抱歉给您带来的不便，对您的心情我们深表歉意。

经核实，本次问题的原因是……我们完全理解您的感受，这也是我们工作中的疏漏。

为表诚意，我们将为您提供：
1. ……
2. ……

同时，我们将在以下方面进行改进：
- ……

再次致以最诚挚的歉意。如您有任何疑问，欢迎随时联系我们 [联系方式]。

XX 客服团队
```

**禁忌**：
- 避免 "如果" 推责（"如果给您造成了困扰"→ 应为"给您造成的困扰"）
- 避免 "但是" 转折（一转折就失去真诚感）
- 避免"系统原因 / 第三方原因"甩锅

## 5. LinkedIn Humblebrag

**核心特征**：自谦的炫耀 + 励志转折 + 鼓舞他人。见 [internet.md](internet.md) 的 LinkedIn 节。

## 6. 银行 / 法律术语体

**特征**：把简单意思包在最曲折、最免责的说法里。

**例**：
- "您的账户有 0 元"
- → "就您名下账户当前余额查询结果而言，目前系统显示为人民币 0 元整。如对该结果存在疑义，敬请携带本人有效身份证件至我行任一营业网点办理相关核实查询手续。"

## 改写注意

- **商务风最忌"热情过度"**。Hi, hope you're doing great today! 在中文邮件里会让收件人警觉。
- **律师函不能编造法条**。原文没有明确法律依据时，用 "相关法律法规"、"适用法律"。
- **客服道歉的核心是"明确 + 具体 + 无推诿"**。空泛的"给您带来不便"远比不上"我们本应在订单生成 2 小时内发货，这次延误了 6 小时"。
- **level 要匹配**。对方是 CEO 用 "Hey" 过于随意；对方是朋友用 "Yours sincerely" 过于生分。
