# 汉语官方体 / 体制内风格

核心特征：**抽象名词化** + **四字结构堆叠** + **弱化主语** + **前置铺垫** + **结论模糊化**。让人读完觉得"说了"但又说不清"说了什么"。

## 共性特征

- **动词 → 名词化**：做 → 开展工作；想 → 研究考虑；看 → 研判
- **多用抽象集合名词**：相关部门、有关方面、各个层面、若干领域
- **四字结构**：统筹推进、精准发力、系统谋划、提质增效
- **连字法**："既 A 又 B"、"以 A 促 B"、"既要 A 也要 B，更要 C"
- **弱化主语 / 用"我们"**："经研究"、"按照工作安排"、"我们认为"
- **数字编号**："一是"、"二是"、"三是"、"第一个"、"第二个"
- **模糊程度副词**：一定程度、较大幅度、明显、显著、初步
- **礼貌模糊化**：存在一定困难、尚需进一步研究、下一步将持续关注

## 子风格

### 1. 体制内汇报风（最常用）

**结构模板**：
```
1. 政治站位 / 背景铺垫（"在 XX 指导下" / "按照 XX 部署"）
2. 工作举措（"一是……二是……三是……"）
3. 取得成效（动词+名词化 + 成果堆叠）
4. 存在问题（"也要清醒看到"+ 模糊弱化）
5. 下一步工作（"下一步，我们将……"）
```

**高频词汇**：
深入学习、统筹推进、精准发力、压实责任、形成合力、久久为功、凝心聚力、守正创新、锚定目标、提质增效、攻坚克难、靶向施策、常态长效、一体推进、协同联动

**例**：
- 原文："这个项目做完了，效果不错。"
- 改写：
  > 一是强化统筹推进。按照既定工作安排，各相关方面凝心聚力、协同联动，高质量推动项目落地实施。
  > 二是取得阶段性成效。项目整体推进平稳有序，预期目标基本达成，为下一阶段工作夯实了基础。
  > 三是下步持续发力。我们将紧盯目标不放松，在总结经验的基础上研究完善相关机制，推动工作再上新台阶。

### 2. 新闻联播风

**结构模板**：时间 + 地点 + 人物 + "强调指出" + 引述内容 + "会议（活动）认为"

**高频句式**：
- "XX 今天在 YY 强调指出……"
- "会议（活动）认为……"
- "各地各部门要切实把思想和行动统一到……"
- "进一步增强……的政治自觉、思想自觉、行动自觉"

### 3. 领导讲话风

**特征**：比汇报风更口语，但仍保留四字结构；喜欢"三个词"并列 + 抑扬顿挫。

**签名句式**：
- "我讲三点意见：第一……第二……第三……"
- "关键是要抓好三个字：X、Y、Z。"
- "这里我强调一下。"
- "讲到底，还是要……"
- "不能有丝毫的松懈"

### 4. 八股 / 材料语

**特征**：堆段子、排比成阵、主语虚化到极致。常见于动员讲话、总结材料。

**例**：
- 原文："要认真对待。"
- 改写："要以更高站位、更宽视野、更实举措，切实做到思想上高度重视、行动上坚决落实、作风上严谨扎实。"

## 改写注意

- **事实只能弱化，不能反转**。"没做完" 可以说成 "正在有序推进"、"已进入收尾阶段"，但不能说"已顺利完成"。
- **数字要保持原样**。如果原文没具体数字，不要编造；用"相关数据"、"一定比例"过渡。
- **不要加入政治口号**除非用户明确要求。现代材料语本身不必加"伟大复兴"等口号。
