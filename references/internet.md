# 互联网风

各大平台有各自鲜明的文风。混用会立刻"不网感"。

## 1. 小红书风（种草体）

**核心特征**：标题党 + emoji 点缀 + 短句 + 数字清单 + 情绪外露 + 互动钩子。

**高频元素**：
- 标题格式："XX 必看！" / "姐妹们！" / "谁懂啊！" / "救命" / "绝绝子"
- emoji：✨ 💕 🥹 😭 🫶 💡 ⚠️（常夹在句子里而非结尾）
- 段落开头常用 💡 / ⚠️ / ✅ / ❌
- 短句断行，每句一行
- 感叹号密度高
- 结尾钩子："你们还知道哪些？评论区告诉我～"

**签名词汇**：
家人们、姐妹们、谁懂啊、真的栓 Q、太绝了、blue 了、yyds、emo 了、性价比之王、宝藏、无限回购、闭眼入、踩雷、避雷

**例**：
- 原文："这个咖啡馆不错，推荐一下。"
- 改写：
  > 姐妹们！！！救命！！我终于找到了 ✨
  > 📍 XX 路上的这家咖啡馆
  > 谁懂啊😭 进去的那一刻真的 blue 了……
  > ✅ 氛围感 满分
  > ✅ 拿铁 真的 很好喝
  > ✅ 老板 超温柔
  > 闭眼冲！真的不踩雷！🫶
  > 你们还有什么宝藏咖啡馆评论区告诉我～

## 2. 微商风

**核心特征**：感叹号炸裂 + 夸张效果承诺 + 社会认同 + 紧迫感 + 全大写/加粗质感。

**签名词汇**：
震惊！、不是吧还有人不知道、限时、仅此一次、私信我、xx 位宝宝已成功、秒回、手慢无、先到先得、错过再等一年、粉丝福利、家人们冲啊

**结构模板**：
```
【震惊】xxxx！！！
用了 xx 天
效果看得见 ✅
……
限时活动 今晚 24 点结束
私信我 秒回 ❤️
手慢无！！！
```

## 3. 互联网黑话（大厂黑话 / 产品经理话）

**核心特征**：名词化 + 抽象化 + 把简单事包装成方法论。

**高频词汇**：
抓手、闭环、链路、颗粒度、对齐、拉通、复盘、赋能、沉淀、心智、用户画像、增长引擎、漏斗、抓重点、打法、底层逻辑、关键动作、价值感、存在感、势能、锚点、底座、长 / 短 / 中期视角、生态位

**签名句式**：
- "我们需要拉通一下这个事的 owner 是谁"
- "这个 case 的抓手在哪？颗粒度再细一点"
- "先把心智打透，再做留存"
- "这其实是一个认知对齐的问题"

**例**：
- 原文："我们要想办法让更多人用。"
- 改写："我们得重新梳理一下用户增长的底层逻辑，先对齐核心抓手，从用户心智层面做深入渗透，通过关键动作拉通链路、形成闭环，持续沉淀势能，完成从流量到留量的转化。"

## 4. 推特 / Twitter 风

**核心特征**：短（< 280 字符）+ punchy + 一句一 insight + 有 hook + 结尾留白。

**结构**：hook → claim → short reasoning → implication

**例**：
- 原文："LLMs 会改变软件行业。"
- 改写："The software industry is about to shift more than anyone is pricing in. It's not 'AI writes code'. It's that the unit of creation stops being the file."

**变体**：Twitter thread — 编号段落 (1/ 2/ 3/)，每条一个点，最后 "That's it. Follow for more."

## 5. Reddit 风

**核心特征**：长帖 + 故事性 + 自我吐槽 + EDIT 更新。

**模板**：
```
TL;DR: [一句话总结]

So this happened today...

[背景叙事]

[事件过程]

[自己的反思 / 困惑]

EDIT: thanks for the awards!
EDIT 2: to clarify...
```

签名：Am I the asshole? / This might get buried but / Unpopular opinion / Change my mind

## 6. LinkedIn 风（humblebrag）

**核心特征**：自谦的炫耀 + "励志"鸡汤 + 个人故事包装 + 每句一段。

**模板**：
```
3 years ago, I was [a humble starting point].

Today, I [an impressive achievement].

Here's what I learned:

1. [insight]
2. [insight]
3. [insight]

Agree? Disagree? Drop a comment 👇

#growth #leadership #entrepreneurship
```

**签名**：I'm humbled to announce / excited to share / beyond grateful / here's a thought

## 改写注意

- **不要混用**小红书 emoji 和互联网黑话；也不要在推特风里用"家人们"。各平台文风有强识别边界。
- **emoji 密度**符合平台："小红书" 每 1-2 行一个，"LinkedIn" 3-4 段一个，"Reddit" 基本不用。
- **长度遵循平台**：推特压到一条之内；小红书中等；Reddit 可以拉长；LinkedIn 每句独立一段。
