# 文学体

## 1. 鲁迅风

**核心特征**：冷峻的反讽 + 一两句看似平淡的短句藏锋 + 对"看客"与"国民性"的洞察 + 破折号和省略号点睛。

**句式习惯**：
- 短句起兴，长句展开："然而，——"、"大约，——"
- 以对某种"现象"的不解开场："我不懂的……"
- "阿 Q 式"的心理转圜：自我安慰与自我欺骗的并置
- 看似闲笔的名词化收束："这大约是 xxx 罢。"

**签名词汇**：
大约、其实、罢、或者、据说、所谓、从来如此、——便是

**例**：
- 原文："这帮人真是糟糕透顶。"
- 改写：
  > 我看着他们一个个庄严地走过，忽然想起"先前"的人也曾这样。大约，人是没有变的，只是名字换过几次罢了。——至于那"糟糕"二字，是我不敢说的；因为我不说，也终究会有人替我说。

### 鲁迅风的"坑"
- 不要滥用"呐喊"、"彷徨"这类明显标签词
- 不要加"吾辈"、"尔等"（那是更早的白话文）
- 关键不在古词，而在**冷眼 + 自嘲 + 对庸众的悲悯**

## 2. 文言文 / 古文风

**核心特征**：单音节词为主 + 虚词 (之/乎/者/也/矣/焉) + 倒装 + 省略主语 + 四字节奏。

**高频虚词**：
之、其、乃、遂、方、既、已、且、然、则、固、诚、盖

**句式**：
- 判断句："XX 者，YY 也。"
- 被动："为 X 所 Y"
- 倒装："何以为之"、"无从措手"
- 感叹："呜呼"、"嗟乎"、"悲夫"

**例**：
- 原文："今天我加班到十一点，很累。"
- 改写："今日伏案至子时，困顿不堪，几欲委顿。嗟乎，勤勉如斯，不知所为何？"

### 文言风的"坑"
- 不要"古风现代混搭"。现代词（电脑、地铁）若必须出现，用**功能等价**的古词代替（"机铎"、"铁车"）或保留但标注。
- 不要只堆"之乎者也"。真正的文言节奏在**动词单字化** + **句法省略**。

## 3. 朋友圈文艺风

**核心特征**：短句 + 断行 + 模糊意象 + 气氛大于事实。

**签名元素**：
- 景物 + 心境并置："风很大，我很想你。"
- 时间标记："晚安 / 晚风 / 夏夜 / 清晨"
- 不说完的句子：省略号收尾
- 引用金句（莫兰迪 / 春上 / 三毛 / 海子 / 昆德拉）

**例**：
- 原文："今天心情不好。"
- 改写：
  > 傍晚六点的光像一块慢慢化掉的糖。
  > 街上人很多，我好像在哪里丢了什么。
  > ——
  > 可能是一些轻一点的东西，比如从前。

## 4. 煽情鸡汤风

**核心特征**：排比 + 名人引用（真假不论）+ 对读者直接称呼"你"+ 结尾升华。

**签名模板**：
```
人生就像 XX，有 A，也有 B。
你以为 ……，其实 ……。
记住：YY 的人，最终都 ZZ。
——致每一个 ……的你。
```

**高频词**：
愿你、总有、终将、不辜负、眉眼含笑、温柔、热爱、生活、拥抱、治愈、遇见、相遇、向阳而生

**例**：
- 原文："坚持下去就会变好。"
- 改写：
  > 你以为光是突然降临的，其实光从来都是一步一步走来的。
  > 那些不被看见的夜里，你蜷在书桌前，以为世界忘了你——
  > 可你忘了，世界只是在等你先不忘了自己。
  > 愿你终将遇见那个，走过长夜之后依然眼里有光的自己。

## 改写注意

- **鲁迅 vs 文言 vs 文艺**是三种完全不同的"古风感"。别混。
- **鸡汤风在专业场合会显得 cringe**。除非用户明确要，否则不要自动往鸡汤走。
- **引用要慎重**。不要伪造名人语录。如果一定要引用，用 "正如有人所说"、"俗话说" 代替具名引用。
